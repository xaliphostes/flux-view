Timer example

We illustrate here how attributes and DOM's children can be plugged to observables.

A count down is started at page loading, and displayed in red when count < 5. Also, depending on the mouse position, the background of the rendered element changes.

When the count down is over the view is removed; in the debug console you can check that all subscriptions are gone.

The source code can be found from the debug panel under sources => webpack:// => . => main.ts
